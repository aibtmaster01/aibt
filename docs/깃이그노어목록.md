# .gitignore 목록

> 프로젝트 루트 `.gitignore` 기준. Git 추적에서 제외되는 파일·디렉터리 목록.

---

## 로그·빌드

| 패턴 | 설명 |
|------|------|
| `logs`, `*.log`, `npm-debug.log*` 등 | 각종 패키지 매니저 디버그 로그 |
| `node_modules` | npm 의존성 |
| `dist`, `dist-ssr` | Vite 빌드 결과물 |
| `*.local` | 로컬 환경 파일 |

## 환경·시크릿

| 패턴 | 설명 |
|------|------|
| `.env`, `.env.*` | API 키 등 비밀값 (`.env.example`만 허용) |
| `**/serviceAccountKey.json` | Firebase 서비스 계정 키 |
| `**/*-firebase-adminsdk*.json` | Firebase Admin SDK 키 |
| `backend/.env`, `backend/.venv/` | 백엔드 환경 변수·가상환경 |

## Python

| 패턴 | 설명 |
|------|------|
| `.venv/`, `venv/`, `__pycache__/` | 가상환경·캐시 |
| `*.py[cod]`, `*.pyo`, `.Python`, `*.so` | 컴파일 산출물 |

## 백엔드 대용량 데이터

| 패턴 | 설명 |
|------|------|
| `backend/BIGDATA/Final_*.json` | 최종 문제 데이터 JSON |
| `backend/BIGDATA/*_final.json` | 최종 산출물 |
| `backend/BIGDATA/*1000*.json`, `*1260*.json` | 대량 문제 JSON |

## 에디터·OS

| 패턴 | 설명 |
|------|------|
| `.vscode/*` (`.vscode/extensions.json` 제외) | VS Code 설정 |
| `.idea`, `.DS_Store` | JetBrains, macOS 메타 |
| `*.suo`, `*.ntvs*`, `*.njsproj`, `*.sln`, `*.sw?` | Visual Studio·vim swap |

## 기타

| 패턴 | 설명 |
|------|------|
| `google-cloud-sdk/` | Google Cloud SDK 설치 디렉터리 (~400MB) |
