# 정책: 유저 플로우

> 서비스 목적, 사용자 여정, 라우트. 예창패 사업계획서용.

---

## 서비스 한 줄 요약

**데이터 자격증(빅데이터분석기사, SQLD, ADsP) 수험생을 위한 AI 기반 모의고사·약점 분석 서비스.**  
기출 풀 기반 맞춤형 회차(Round 4+), 약점 공략(Round 6+), 3차원 통계(개념·유형·과목·태그)로 데이터 기반 합격 가이드를 제공.

---

## 전체 사용자 여정

```
[비로그인] 홈 → 무료 진단 → 모의고사 목록 → 퀴즈(20문항) → 로그인/결제 유도
[로그인 무료] 홈/마이페이지 → 1·2회차 풀기 → 3회차부터 결제 유도
[로그인 유료] 전 회차 + Round 4+ 맞춤형 + 약점 공략 → 결과·대시보드
[결제] 회차 선택 → 결제 완료 → 이용권 → 시험당일 12:00 KST 이후 활성화
```

## 라우트

| 경로 | 역할 |
|------|------|
| `/` | Home — 진단 시작, 로그인/결제 유도 |
| `/login` | 로그인 모달 오픈 (별도 페이지 없이 모달) |
| `/mypage` | 나의 학습 대시보드 (로그인 필수) |

| `/exam-list` | 모의고사 회차 목록 |
| `/quiz` | 퀴즈 진행 |
| `/result` | 시험 결과 |
| `/checkout` | 결제 (모달로 표시) |
| `/admin` | 관리자 전용 |

## 주요 시나리오

### A. 비로그인 → 모의고사 → 로그인/결제
1. 홈에서 "무료 진단" 또는 자격증 카드 클릭 → `/exam-list`
2. 회차 선택 → `/quiz` (게스트 20문항 제한)
3. 20문항 제출 시 "로그인하면 이어풀 수 있어요" → `/login` (모달)
4. 로그인 성공 시 퀴즈 이어하기 또는 결제로 이동

### B. 로그인 무료회원
- 1·2회차: 풀기 가능
- 3회차 이상: 잠금, "열공모드로 합격권에 진입하세요" → 결제 유도

### C. 로그인 유료회원
- Round 1~3 고정형, Round 4+ 맞춤형, Round 6+ 약점 공략 전체 이용
- AI 해설 열람 가능
