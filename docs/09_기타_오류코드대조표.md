## 7K2M-A9P1 (ERR_FIREBASE_PERMISSION) 상세

게스트/회원이 **문제풀기** 시 "Missing or insufficient permissions" 발생 시:

1. **브라우저 콘솔(F12)** 로 실패 지점 확인:
   - `[getQuestionsForRound] getDoc(static_exams) 실패` → static_exams 규칙·App Check
   - `[getQuestionsForRound] fetchQuestionsFromPools 실패` → collection group 규칙·인덱스

2. **배포 프로젝트 일치**  
   `firebase use aibt-99bc6` → `firebase deploy --only firestore:rules`

3. **Firestore 규칙**  
   `certifications/{path=**}` → `allow read: if true` (게스트 포함) 확인.

4. **App Check (가장 흔한 원인)**  
   Firestore에 App Check 적용 시 토큰 없는 요청은 전부 거부.  
   - 테스트: Monitor 모드 또는 Enforce 해제  
   - 운영: 웹 앱 App Check 등록(reCAPTCHA v3 등) 후 토큰 붙이기
