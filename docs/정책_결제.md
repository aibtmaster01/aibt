# 정책: 결제 · 장학금 · 쿠폰 제도

> 이용권 기반 수익 모델과 장학금/쿠폰 할인 정책.

---

## 1. 이용권(Pass) 모델

| 항목 | 내용 |
|------|------|
| **상품** | 자격증별 **이용권(Pass)** — "제 N회 시험 대비" 회차 단위 |
| **기본 정가** | 39,000원 |
| **상태** | `pending` → `active` → `expired` / `cancelled` |
| **활성화 시점** | 해당 회차 시험당일 **12:00 KST 이후** 앱 진입 시 자동 전환 |
| **주문 취소** | 시험당일 **12:00 이전**에만 가능 |
| **회차 변경** | D-7 이상, 맞춤형 미시행, 1회 한정 |

### Firestore 구조

```
users/{uid}/
  passesByCert: {
    [certCode]: {
      round: number,
      examDate: string,        // "2026-04-08"
      status: "pending" | "active" | "cancelled",
      roundChangeUsed: boolean
    }
  }
```

### 관련 코드

| 파일 | 역할 |
|------|------|
| `src/pages/Checkout.tsx` | 결제 페이지 (현재 PG 미연동, 2초 시뮬레이션) |
| `src/services/authService.ts` | `setPaymentComplete`, `cancelPass`, `membershipsToUserFields` |
| `src/pages/AccountSettings.tsx` | 이용권 표시·주문취소·회차변경 |

## 2. 장학금·할인 정책 (구현 예정)

| 정책 | 조건 | 혜택 | 마케팅 네이밍 |
|------|------|------|---------------|
| **① 첫 진입** | 해당 자격증 최초 결제 | 10,000원 할인 → 29,000원 | **첫 수강 1만 원 지원금** |
| **② 합격 보상** | 합격 후 D+1 설문 "합격했어요" | 실결제 50% 환급 (14,500원) | **합격 축하 반값 장학금** |
| **③ 재도전** | 불합격 후 재수강 | 9,900원 파격 (75%↓) | **눈물의 9,900원 패스** |
| **④ 교차 학습** | 타 자격증 완료 → 새 자격증 | 10,000원 쿠폰 → 29,000원 | **커리어 빌드업 응원금** |

### 정책별 상세

- **① 첫 진입**: 결제 단계에서 자동 할인 적용. 카피: "책 한 권 값으로 AI 과외를 시작하세요."
- **② 합격 보상**: 합격 확인 후 환급 (계좌 입금 / 포인트 / 다음 결제 차감 중 결정 필요). 카피: "합격의 기쁨에 14,500원 용돈까지 얹어드립니다!"
- **③ 재도전**: 불합격 후 재결제 시 자동 적용. 카피: "커피 두 잔 값으로 지난번 오답 데이터를 살려 재도전하세요."
- **④ 교차 학습**: 자격증 A 완료 → B용 쿠폰 발급. 카피: "ADsP 끝내셨나요? SQLD 도전을 위한 1만 원 쿠폰이 도착했습니다."
