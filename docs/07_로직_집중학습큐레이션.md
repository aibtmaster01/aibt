# 로직: 집중학습 큐레이션 (과목/유형/개념)

> 과목 강화, 취약 유형 집중, 취약 개념 집중 — 각 50문항 선정 로직.

---

## 진입 흐름

1. 마이페이지에서 버튼 클릭 (과목 강화 / 취약 유형 / 취약 개념)
2. 모드 선택 모달 (학습 모드 / 시험 모드)
3. 3초 준비 오버레이
4. 50문항 큐레이션 → 퀴즈 직행

---

## 과목 강화 (fetchSubjectStrengthTraining50)

- **대상**: 선택한 과목의 문항 50개.
- **조건**: 모의고사 1회 이상 응시, 해당 과목 문제 50개 이상.
- **미충족 시**: "아직 충분한 데이터가 쌓이지 않았어요" 안내.
- **정렬**: subject_number 순.

---

## 취약 유형 집중 (fetchWeakTypeFocus50)

- **우선순위** (problem_type_stats proficiency 낮은 순):
  1. 가장 취약 유형의 **틀린 문제** (최대 50)
  2. 가장 취약 유형의 **맞춘 문제**
  3. 가장 취약 유형의 **난이도 스캐폴딩** (proficiency 기반)
  4. 두 번째 취약 유형으로 동일 반복
- **0문항 시**: "모의고사를 1회 이상 응시한 뒤 이용해 주세요."
- **정렬**: subject_number 순.

---

## 취약 개념 집중 (fetchWeakConceptFocus50)

- **대상**: core_concept_stats 또는 sub_core_id_stats proficiency 낮은 개념.
- **선정**: 틀린 문제 우선, 부족분은 난이도 스캐폴딩.
- **정렬**: subject_number 순.
