## 필드 설명

- **구독 자격증**: `subscriptions` — 해당 자격증의 회차 목록/대시보드 이용 대상.
- **결제 완료(cert)**: `paidCertIds` — 결제한 자격증 ID. 프리미엄 회차(3회차 이상, 약점 공략 등) 이용 가능.
- **만료(cert)**: `expiredCertIds` — 결제했으나 수강 기간이 만료된 자격증. 재수강/결제 유도 대상.
