|------|------|------|-------------|
| **① 첫 진입** | 해당 자격증 최초 결제 | 10,000원 할인 → 29,000원 | "첫 수강 1만 원 지원금" |
| **② 합격 보상** | 합격 후 D+1 설문 "합격했어요" | 실결제 50% 환급 | "합격 축하 반값 장학금" |
| **③ 재도전** | 불합격 후 재수강 | 9,900원 파격 (75%↓) | "눈물의 9,900원 패스" |
| **④ 교차 학습** | 타 자격증 완료 후 새 자격증 | 10,000원 쿠폰 | "커리어 빌드업 응원금" |

## 상세

### ① 첫 진입
- 결제 단계에서 할인 코드 또는 자동 적용
- UI: "체감가 29,000원" 표시

### ② 합격 보상
- D+1 설문 응답 후 환급 처리
- 환급액 = 실결제액 × 0.5

### ③ 재도전
- D+1 설문 "아쉬워요" 응답 등으로 재도전 대상 식별
- 재결제 시 9,900원 자동 적용

### ④ 교차 학습
- 자격증 A 합격/완료 시 자격증 B용 10,000원 쿠폰 발급
- `users/{uid}/coupons` 또는 `available_coupons` 저장 예상
