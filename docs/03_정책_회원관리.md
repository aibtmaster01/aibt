# 정책: 회원 관리

> 가입, 탈퇴, 인증, 멤버십·구독 관리. 실제 코드 기준 정리.

---

## 등급 정의 (examService.getUserGradeForCert)

| 등급 | 조건 |
|------|------|
| **Guest** | user === null 또는 subscriptions에 certId 없음 |
| **Free** | 구독은 있으나 paidCertIds에 certId 없음 |
| **Premium** | paidCertIds에 certId 있으며 expiredCertIds에 없음 |
| **Expired** | paidCertIds에 certId 있으나 expiredCertIds에 포함 |
| **Admin** | isAdmin === true → 항상 Premium |

---

## 접근 제어 (checkExamAccess)

| 등급 | Round 1 | Round 2 | Round 3 | Round 4+ |
|------|---------|--------|--------|----------|
| Guest | O (20문항 제한) | X | X | X |
| Free | O | O | X | X |
| Premium | O | O | O | O |
| Expired | O (오답노트만) | X | X | X |

## 인증·회원 관련 코드

| 파일 | 역할 |
|------|------|
| `src/services/authService.ts` | 로그인/회원가입/로그아웃, 비밀번호 재설정, memberships → paidCertIds/expiredCertIds 변환 |
| `src/contexts/AuthContext.tsx` | 전역 로그인 상태(user, loading), login/logout 제공 |
| `src/components/LoginModal.tsx` | 로그인·회원가입 모달 (이메일/비밀번호, Google) |
| `src/pages/AccountSettings.tsx` | 계정 설정, 이용권 표시, 주문 취소 |

## Firestore 경로

| 경로 | 용도 |
|------|------|
| `users/{uid}` | 프로필, subscriptions, passes, passesByCert, paidCertIds, expiredCertIds |
| `memberships` (adminService 연동) | 관리자용 멤버십 조회 |
| `admin_users` | 관리자 권한 |
