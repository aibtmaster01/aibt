## 정책별 상세

### ① 첫 진입 (첫 수강 1만 원 지원금)
- **조건**: 해당 자격증 최초 결제
- **혜택**: 10,000원 할인 → 체감가 29,000원
- **적용 시점**: 결제 단계 (할인 코드 또는 자동 적용)

### ② 합격 보상 (합격 축하 반값 장학금)
- **조건**: 시험 합격 후 D+1 설문 등에서 "합격했어요" 응답
- **혜택**: 실 결제액 50% 환급 (예: 29,000원 결제 시 14,500원 페이백)
- **적용 시점**: 합격 확인 후 환급 처리

### ③ 재도전 (눈물의 9,900원 패스)
- **조건**: 불합격 후 재수강 (D+1 설문 "아쉬워요" 응답 등)
- **혜택**: 9,900원 파격 할인 (정가 39,000원 대비 75%↓)
- **적용 시점**: 재결제 시 할인 코드 또는 자동 적용

### ④ 교차 학습 (커리어 빌드업 응원금)
- **조건**: 다른 자격증 합격/완료 후 새 자격증 구독
- **혜택**: 10,000원 할인 쿠폰 → 다음 자격증 29,000원
- **적용 시점**: 새 자격증 결제 시 쿠폰 적용
